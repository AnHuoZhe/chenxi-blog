# 视频生成流水线 —— 需求文档 V3

## 项目目标

命令行工具。喂一期文案+对应音频，输出横屏竖屏两版视频。

## 输入格式

```
ep1/
├── script.txt          ← 文案（Ai发展文案.docx格式）
├── audio/
│   ├── 01.wav          ← 综述段
│   ├── 02.wav          ← 分段第一段
│   ├── ...
```

script.txt 结构：
```
第一期：《AI的四次劫难》
一、综述
[口播正文]

二、分段
第一劫：规则写不完。
[口播正文]

第二劫：网络做不深。
[口播正文]
...

三、总结和下期预告
[口播正文]

四、自我介绍
我是晨汐。一个用唯物辩证法给你拆AI的人。
```

## 输出

```
ep1/output/
├── horizontal/              ← 横屏 1920×1080（B站）
│   ├── ep1-horizontal.mp4   ← 带章节标记
│   ├── slides/01.png
│   └── subtitles/01.srt
└── vertical/                ← 竖屏 1080×1920（抖音/快手/小红书）
    ├── ep1-vertical.mp4
    ├── slides/01.png
    └── subtitles/01.srt
```

## 处理流程

### 1. 解析脚本
读 script.txt → 按 `一、综述` `二、分段` `三、总结` `四、自我介绍` 切段 → 分段内的子标题（如"第一劫：规则写不完"）作为独立段 → 输出段列表 → 校验与 audio/ 文件数匹配

### 2. 生成PPT画面
用 Pillow 绘制，横屏1920×1080 + 竖屏1080×1920：
- 综述段 → 大标题"第X期" + 副标题
- 分段 → 大字标题（如"第一劫"）+ 副标题（如"规则写不完"）
- 总结段 → "总结" + 下期预告精简版
- 自我介绍 → "我是晨汐" + 固定文案
- 深色背景(slate-950)，白色文字，pink-500强调色

### 3. 音频留白处理
用 pydub 在每段音频末尾拼接2秒静音（段间留白让观众思考）

### 4. 字幕生成
用 OpenMontage 自带的 WhisperX 逐段生成词级SRT字幕

### 5. 视频合成
用 OpenMontage / Remotion 将画面+音频+字幕合成为逐段MP4，拼接为完整视频，嵌入章节标记

### 6. 多格式输出
- 横屏 1920×1080 → B站、YouTube、视频号
- 竖屏 1080×1920 → 抖音、快手、小红书

## 外部依赖

已装（不需要新装）：
- OpenMontage → 视频合成 + 字幕生成（含Remotion、WhisperX、FFmpeg）
- GPT-SoVITS → 音频生成（本项目外，用户手动使用）

需新装：
- pydub → 音频静音插入（`pip install pydub`）
- Pillow → PPT画面绘制（`pip install Pillow`）

## 段间留白

默认2秒。画面定格在上段最后一帧，音频静音。可配置。

## 进度输出

```
解析脚本完成，共7段
PPT生成中 3/7...
字幕生成中 5/7...
合成中 6/7...
输出完成：horizontal/ep1-horizontal.mp4
输出完成：vertical/ep1-vertical.mp4
```

## 不做的

- 不做AI自动切分音频（用户在GPT-SoVITS里逐段生成）
- 不做弹幕飘过效果（只有底部字幕）
- 不自动上传平台
