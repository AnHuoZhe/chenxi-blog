# AI发展第一期 · 视频制作需求

## 项目路径
工作目录：`E:\work\word\study\ai-video-ep1\`

## 你的准备工作
1. 在GPT-SoVITS里逐段生成音频，保存到 `E:\work\word\study\ai-video-ep1\audio\`
2. 文件命名：`01.wav, 02.wav, 03.wav...` ——按段号
3. 每段对应的文案文本

## 全部丢给Codex，一次性跑三条流水线：

### 一、字幕生成

对每个音频文件用 whisper 生成SRT字幕：
```
whisper audio/01.wav --model small --language zh --output_format srt --output_dir subtitles/
```
产出：`subtitles/01.srt, 02.srt...`

whisper用 `openai-whisper` 包，pip install。model选small（中文够用，快）。

### 二、PPT画面生成

对每段文案生成一页HTML幻灯片，参数：
- 深色背景（#0f172a slate-950）
- 大字标题（text-5xl font-bold）
- 3-5个关键词用标签排列（pink-500背景色标签）
- 底部细字标注"晨汐 · AI发展"
- 1920×1080分辨率

产出 `slides/01.html, 02.html...`

用Puppeteer或无头浏览器截图导出为PNG：`slides/01.png, 02.png...`

如果Puppeteer太重，改用Python的 `imgkit` 或 `playwright`。只要能截图就行。

### 三、合成视频

用FFmpeg把每段——画面(PNG) + 音频(WAV) + 字幕(SRT) ——合成MP4：

```bash
ffmpeg -loop 1 -i slides/01.png -i audio/01.wav -vf "subtitles=subtitles/01.srt:force_style='FontSize=28,PrimaryColour=&H00FFFFFF,OutlineColour=&H00000000,Outline=2'" -c:v libx264 -tune stillimage -c:a aac -b:a 192k -pix_fmt yuv420p -shortest output/01.mp4
```

所有段合成完后，合并：
```bash
ffmpeg -f concat -safe 0 -i concat.txt -c copy output/ep1-final.mp4
```

concat.txt格式：
```
file 'output/01.mp4'
file 'output/02.mp4'
...
```

### 四、验证并提交
